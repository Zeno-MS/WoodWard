## The Bottom Line

The core reporting question is whether VPS publicly described Amergis as an approximately $3 million contract while the district's own records support a much larger actual-payment figure in the same fiscal year, and whether that gap was tracked, explained, or meaningfully overseen.

## What This Package Does Not Assume

- It does not assume you will run code or rebuild the payment dataset.
- It does not assume every open question has already been answered.

It is meant to help you verify the central story and decide where to report next.
