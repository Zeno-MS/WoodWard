# Ready-to-File Public Records Requests
## Vancouver Public Schools, RCW 42.56

These requests are designed to be filed by anyone: a journalist, civic watchdog, union member, parent, or resident. Each request is narrow, specific, and high-yield. No specialized tools or background knowledge required.

## File This One First

Start with **Request 2: The $3 Million vs. $10.97 Million Variance**.

Why: it goes directly to the clearest reporting question. If responsive records exist, they may show whether district officials internally tracked, explained, or briefed anyone about the gap between the board-facing approximate $3 million figure and the much larger actual Amergis payments. If no responsive records exist, that absence is itself meaningful.

## Suggested Filing Sequence

1. File Request 2 first.
2. File Request 1 second if you need the full contract terms, amendments, or rate sheets.
3. File Request 4 if the approval chain becomes the story.
4. File Requests 3, 5, 6, and 7 if you are expanding beyond the main hook into oversight, staffing strategy, or broader fiscal management.

## When To Use Records Requests Versus Comment Requests

Use a records request when you want documents, internal analyses, approval records, or written briefings.

Use a comment request when you want an explanation, defense, justification, or present-tense position from an official.

In plain English: ask for records when the answer should exist on paper. Ask for comment when the answer is a person explaining or defending a decision.

**Filing instructions:** Submit via VPS public records request portal or email to the district's Public Records Officer. Cite RCW 42.56 (Washington Public Records Act). No reason or justification is required.

## Request 1: The Amergis/Maxim Contract and All Amendments

**Priority:** High

**Best use case:** File this if you want the full current contract picture, especially amendments, rate sheets, and attachments not visible in public board materials.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> All versions of the Master Service Agreement, Client Services Agreement, or any other contract between Vancouver Public Schools and Amergis Healthcare Staffing, Inc. (formerly Maxim Healthcare Services, Inc.) currently in effect or in effect at any point during FY2024-25 and FY2025-26, including all amendments, addenda, attachments, rate sheets, rate schedules, and scope-of-work documents.

**Why this matters:** The contract terms (auto-renewal, uncapped rates, conversion fee, no spending cap) are the structural core of the vendor relationship. The May 2021 MSA is partially available through BoardDocs, but amendments and current rate sheets are not.

## Request 2: The $3 Million vs. $10.97 Million Variance

**Priority:** Highest first request

**Best use case:** File this first if your main reporting question is how the board-facing approximate figure relates to the district's actual payments.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> Any internal variance analysis, board briefing, memorandum, email, email summary, finance report, budget amendment, or other document created between July 1, 2024 and the present discussing, analyzing, or referencing the difference between the approximately $3,000,000 cost estimate presented to the Board of Directors for the Amergis Education contract (approved July 9, 2024) and the actual amount paid to Amergis Healthcare Staffing, Inc. during FY2024-25.

**Why this matters:** The board approved an estimated $3M. Actual payments exceeded $10.97M. If internal documents flagging this variance exist, they show whether anyone inside the district noticed. If they don't exist, that is itself a finding about the adequacy of financial monitoring.

## Request 3: Object 7 Vendor-Level Spending Breakdowns

**Priority:** High if you are expanding into the broader fiscal-management story

**Best use case:** File this if you want to know whether the district internally tracks outside-services spending by vendor and whether the board ever saw that detail.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> Any vendor-spend report, purchased-services dashboard, monthly finance packet, internal report, or board briefing breaking down Object 7 (Purchased Services) spending by vendor or by subcategory for FY2023-24, FY2024-25, and FY2025-26 (year to date). This includes any report identifying individual vendors and amounts within Object 7.

**Why this matters:** Object 7 actual spending exceeded the budget by $10.6M in FY2024-25. Staffing vendors accounted for 28.16% of actual Object 7 spending. Whether the district tracks this at the vendor level, and shares that tracking with the board, is a governance question.

## Request 4: Contract Approval Chain

**Priority:** High if you need to identify responsible officials

**Best use case:** File this if the story is shifting toward who signed off, who renewed the agreement, and who accepted rate changes.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> Any document identifying who approved, reviewed, renewed, or monitored the Amergis Healthcare Staffing and/or Maxim Healthcare Services agreements, including but not limited to routing sheets, signature pages, approval forms, delegated authority records, and any record showing which district official(s) authorized contract renewals or rate-increase acceptances.

**Why this matters:** The contract auto-renews annually. Vendor rates can increase on 30-day notice. Someone inside the district accepts these terms each cycle. This request identifies who.

## Request 5: Recruitment and Vacancy Data

**Priority:** Medium to high

**Best use case:** File this if you want to test the district's likely defense that vendor dependence was driven by hiring difficulty rather than policy choice.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> Any recruitment and vacancy analysis, workforce planning report, time-to-fill report, vacancy count, exit interview summary, or retention analysis for special education teachers, paraeducators, speech-language pathologists, school psychologists, school nurses, and behavior staff (including BCBAs and behavior technicians) from FY2022-23 through the present.

**Why this matters:** Every unfilled district position becomes a billing opportunity for a staffing vendor. If the district tracks vacancies and time-to-fill, those records show whether the recruitment pipeline is improving or whether vendor dependence is structural.

---

## Request 6: Conversion Analysis

**Priority:** Medium

**Best use case:** File this if you want to know whether the district ever seriously evaluated moving agency workers into direct employment or measured the financial impact of the contract's conversion-fee structure.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> Any analysis, report, memorandum, or discussion document evaluating the conversion of agency-placed workers into permanent district employment, including any reference to placement-fee or conversion-fee provisions in staffing vendor contracts, and any cost-benefit analysis comparing vendor rates to direct-hire costs.

**Why this matters:** The Amergis MSA includes a 30% conversion fee penalizing the district for hiring workers the vendor has placed. Whether the district has ever analyzed the cost of this provision, or evaluated a direct-hire alternative, reveals whether the vendor relationship is a deliberate strategy or a path-dependent default.

---

## Request 7: Emergency Borrowing and Object 7 Overages

**Priority:** Medium to high

**Best use case:** File this if you are trying to connect the district's cash-flow stress to its outside-services spending and determine whether officials discussed that link internally.

> Pursuant to RCW 42.56, I request copies of the following records:
>
> Any board briefing, executive briefing, finance committee presentation, internal memorandum, or email discussing registered warrants, emergency apportionment advances, or Object 7 (Purchased Services) budget overages that references staffing vendors, Amergis, Maxim, or contracted staffing, from FY2023-24 through the present.

**Why this matters:** The district borrowed emergency funds three years running. If any internal discussion linked the borrowing to uncapped vendor spending, that connection should be on the record. If no such discussion exists, that is itself an indicator of the governance gap.

## Notes for Filers

- **You do not need to explain why you want these records.** RCW 42.56 does not require a stated purpose.
- **The district must respond within five business days** with either the records, an estimate of time needed, or a denial citing a specific exemption.
- **If a request is denied**, ask for the specific RCW exemption cited. Staffing contracts, vendor payment records, and board briefings are generally not exempt.
- **Keep a copy of your request and the district's response.** This creates a paper trail regardless of what the district produces.
- **You can file these requests independently.** Each request stands alone. File one or file all seven.

## Practical Reporter Notes

- If you are trying to move fast, do not file all seven at once unless you are prepared to track them carefully.
- If you only have time for one comment request and one records request, pair a right-of-reply question with **Request 2**.
- If the district says no records exist for a major variance or approval question, that answer may itself be newsworthy.
- If the district gives a long time estimate, ask whether it can produce records on a rolling basis.
