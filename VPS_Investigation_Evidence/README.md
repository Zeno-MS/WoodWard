# Vancouver Public Schools Investigation: Journalist Handoff Package

## What is this?
This is a self-contained, fully verified evidence package documenting a multi-year investigation into Vancouver Public Schools' (Clark County, WA) dependency on private healthcare staffing agencies during a severe austerity cycle.

The thesis is mathematically proven using public documents: To close a $35M deficit, the district cut 217 public jobs while simultaneously paying a $13.3M premium to private corporate staffing agencies during the same fiscal year. The burden of the cuts fell asymmetrically on low-paid support staff, while the top-heavy executive architecture remained largely intact.
