## How to Find Object 7 Data

### Method 1: Direct District Access
1. Visit each district's budget page (links above)
2. Download the F-195 Budget Summary for each year (2022-23, 2023-24)
3. Locate the expenditure section organized by "Object" codes
4. Find "Object 7: Purchased Services"
5. Record the total amount

### Method 2: OSPI SAFS Reports
1. Go to: https://ospi.k12.wa.us/policy-funding/school-apportionment/safs-report
2. Select "School Year" dropdown
3. Select "Report Type" (likely "F-195" or "Expenditures by Object")
4. Select specific district or download all districts
5. Extract Object 7 data for both districts

### Method 3: Washington State Report Card
1. Go to: https://reportcard.ospi.k12.wa.us/
2. Search for "Evergreen School District" (Clark County)
3. Navigate to Finance section → "Actual Expenditure by Accounting Codes"
4. Look for Object 7 data for 2022-23 and 2023-24
5. Repeat for "Tacoma School District"

### Method 4: Direct Public Records Request
If documents are not readily accessible online:
- **Evergreen:** Email BudgetQuestions@EvergreenPS.org
- **Tacoma:** Email CFO@tacoma.k12.wa.us
- **OSPI:** Email SAFS@k12.wa.us

Request: "F-195 Budget Summary reports for school years 2022-23 and 2023-24, specifically the Object 7 (Purchased Services) expenditure totals."
